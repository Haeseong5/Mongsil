# <center> **Mongsil**</center>  
몽실은 하루의 감정을 솔직하게 기록할 수 있는 감정일기장 앱입니다.
두루뭉술한 무난한 하루를 보낸 분도, 파란만장한 하루를 겪은 분도 누구나 자유롭게 사용할 수 있도록 다양한 감정과 어느 시간때든 자유롭게 감정을 기록할 수 있도록 타임라인을 제공합니다.
